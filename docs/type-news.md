# Type 1 — News Posts (RSS rewrite) (stub)

От Дарии: «ИИ собирает инфоповоды, сокращает статьи, пишет короткие TG-форматы. Я должна собрать список источников где ИИ будет смотреть, анализировать, собирать сок.»

**Статус:** 🟦 stub. Заполнить **последним** — самый сложный из 5 (требует мониторинга, дедупа, anti-fabrication).

## Открытые вопросы — нужно решить с Дарией

1. **Список источников.** Дария обещала собрать. Форматы:
   - **RSS** (легко) — `affiliateguide.com/feed/`, `partnerkin.com/feed/`, `vc.ru/digital/rss`, etc.
   - **TG-каналы конкурентов** (сложно) — Bot API **не читает чужие каналы**. Нужен либо:
     - User-аккаунт через Telethon / Pyrogram (Python-only — нарушает Java-convention SMM)
     - Public RSS-мирроры этих каналов (если есть)
     - Скрапинг `t.me/s/<channel>` preview-страницы (хрупко)
   - **Сайты без RSS** — нужен HTML-scraper
2. **Частота poll** — каждые 30 мин достаточно для медиа-новостей
3. **Фильтр релевантности** — как отличить affiliate/арбитраж-новость от общей IT-новости?
4. **Дедупликация** — одна новость в 3 источниках одновременно: посчитать одной?
5. **Anti-fabrication** — рерайт **не должен** искажать факты. Особенно числа, имена, даты, регуляторы.

## Caption (предварительно)

```
🚨 <b>{hook_line}</b>

{2-3 короткие строки с фактами из источника}

Источник: {publication_name}, {date}

#новости #арбитраж
```

Длина 200-500 символов (минималистичный новостной формат TG).

## Slides (предварительно)

- **slide_01** — Hook: emoji + headline (one line) + date
- **slide_02** (опционально) — Key facts (3 bullets с цифрами)
- **slide_03** (опционально) — Context / impact / what to do

Большинство news-постов — 1 slide (hook) или text-only. Карусели — только когда есть конкретные числа или сравнения для визуализации.

## Pipeline (набросок)

```
news_sources (DB) — список Дарии
       ↓
NewsAggregator (cron каждые 30 мин)
  - poll RSS
  - dedup по URL hash + fuzzy match заголовков
  - сохранить в news_items_raw
       ↓
NewsScorer
  - relevance filter (gpt-4o-mini классификатор: affiliate=yes/no)
  - оставить top-5 в день
  - выкинуть свои traffnews ссылки
       ↓
NewsDirector (gpt-4o)
  - читает 1 item → рерайт под TG
  - **STRICT anti-fabrication** — все числа/имена/даты из source
  - 3-block (HOOK / FACTS / CTA)
       ↓
SlideExecutor (1-3 slides)
       ↓
Bundle: news_<ts>_<slug>_<hash>/
```

## Anti-fabrication HARD RULE (для news особенно)

Этот тип постов **самый рискованный** по легальности — мы рерайтим чужие новости. Жёсткие правила:

1. **Никаких добавочных утверждений** — если в источнике написано «X запустил Y», нельзя написать «X запустил Y и теперь Z готовится тоже». Только то что в источнике.
2. **Числа — точно как в источнике.** «$50M» в источнике = «$50M» в посте. Не «более $50M», не «около 50», не «50+».
3. **Имена и даты** — verbatim. Никаких упрощений.
4. **Источник всегда указывать** — «по данным {publication}» / «{publication} сообщает» — для distance + atribution.
5. **Запрет на причинно-следственные цепочки** которых нет в источнике («это приведёт к Y», «регуляторы теперь обратят внимание»).

Это правило **уже** есть в `D:\Prog\SMM\CLAUDE.md` (секция «NO fabricated facts»). Для news-pipeline применяется ещё строже.
